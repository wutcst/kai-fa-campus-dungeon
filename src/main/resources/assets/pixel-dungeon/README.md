# Pixel Dungeon Asset Staging

This directory is reserved for Pixel Dungeon assets that may be reused by
Campus Dungeon after the GPL reuse boundary is reviewed.

No upstream assets are imported in this slice.

When assets are added here, each PR must record:

- the original Pixel Dungeon upstream path;
- whether the asset was copied verbatim or modified;
- how the asset is used by Campus Dungeon;
- confirmation that `THIRD_PARTY_NOTICES.md` and
  `licenses/pixel-dungeon/GPL-3.0.txt` remain accurate.
